March 2021 Wallpaper Generator

How it Works:

Play with each of the parameters to create a custom computer wallpaper for March 2021!

First, decide your font. Then play with colors and star shapes.

Once you're happy with the parameter settings, uncheck the description display.

Click to redraw the randomly generated star pattern until you're happy with how it looks. When you're finished, press any key to download your custom wallpaper as a .png file, set to your computer, and enjoy!

Parameter Description

  • fontFamily: chose the font family of the text
  • points: chose the number of points on the stars
  • stroke: stroke thickness of the stars
  • radius 1: size range control 1
  • radius 2: size range control 2
  • starsColor: color picker for stars
  • backgroundColor: color picker for background
